# Imagens do IT-214

## Arquivos necessários

### luiz.jpg
Adicionar a foto de Luiz Henrique Tozi nesta pasta como `luiz.jpg`.

A imagem deve ter dimensões quadradas (recomendado 300x300px ou superior) para melhor resultado no avatar circular.

### ita-logo.svg
Logo do ITA já incluído em `ita-logo.svg`.

Se desejar usar uma imagem diferente do logo do ITA, substitua este arquivo mantendo o mesmo nome.

---

**Nota:** O arquivo HTML referencia automaticamente as imagens em `img/luiz.jpg` e `img/ita-logo.svg`.
